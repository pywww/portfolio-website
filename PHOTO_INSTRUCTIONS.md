# 📸 如何添加你的个人照片

## 步骤：

### 1. 准备照片文件
- 找到你的照片文件（就是那张在咖啡馆的照片）
- 文件格式：JPG 或 PNG
- 建议尺寸：至少 800x1000 像素

### 2. 重命名照片
将照片重命名为：`portrait.jpg` 或 `portrait.png`

### 3. 放入项目文件夹
将 `portrait.jpg` 文件复制到：
```
src/assets/portrait.jpg
```

完整路径应该是：
```
portfolio/
  └── src/
      └── assets/
          └── portrait.jpg  ← 你的照片放这里
```

### 4. 完成！
代码已经更新好了，会自动读取这个照片。
